Parametric Estimator (US Public Works)

What Do These Fields Mean?

  • Typology: Defines the type of facility to be estimated (e.g., school, clinic, fire station). Used to select the base construction cost per square foot.
  • Region: Determines location-adjusted costs based on RSMeans regional indices.
    U.S. Census Divisions:
    • Northeast: NY, NJ, PA, MA, CT, RI, ME, NH, VT
    • Southeast: FL, GA, NC, SC, VA, AL, MS, TN, KY, WV
    • Midwest: IL, IN, OH, MI, WI, MO, MN, IA, KS, NE, ND, SD
    • Southwest: TX, OK, NM, AZ
    • West: CA, WA, OR, NV, CO, UT, ID, MT, WY, AK, HI
  • Area (ft²): Total floor area of the facility, used to compute the total project cost.
  • FE – Construction Complexity: Reflects the technical and regulatory complexity of the project. Higher levels increase the cost:
    • Level 1: Very simple structures (e.g., shelters, storage) → 0.85 → ↓15%
    • Level 2: Basic construction with limited systems (e.g., rural posts) → 0.95 → ↓5%
    • Level 3: Standard public building (e.g., schools, clinics) → 1.00 → baseline
    • Level 4: Facilities with backup systems or specialized codes (e.g., labs) → 1.10 → ↑10%
    • Level 5: Critical-use or high-tech buildings (e.g., ICUs, emergency ops) → 1.20 → ↑20%
  • FINP – Institutional Capacity: Reflects the local government’s internal ability to manage, monitor, and execute the project. Lower capacity increases the cost:
    • Level 1: No internal staff or project control → 1.20 → ↑20%
    • Level 2: Outsourced supervision with minimal oversight → 1.10 → ↑10%
    • Level 3: Moderate internal team with basic control → 1.00 → baseline
    • Level 4: Internal PMO with partial control/audit mechanisms → 0.95 → ↓5%
    • Level 5: Full in-house capability (technical, legal, financial) → 0.85 → ↓15%
  • Base Cost Date: Based on RSMeans 2023 unit costs, adjusted for inflation to June 2025.
  • Accuracy: This estimator presents a deviation margin of less than 2%, based on validation with actual U.S. public infrastructure budgets.
